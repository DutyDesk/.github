### What DutyDesk does

Look up any HTS or TARIC code and get the total duty you'd actually pay: base rate, Section 301 and 232 additions, antidumping/countervailing duties, safeguard measures, and preferential trade programs, all calculated into one number.

- **US and EU coverage.** Every tariff code in both schedules, updated from official government sources.
- **Total landed duty.** Not just the base rate. Trade actions, fees, and preferential programs included.
- **Product classification.** Describe a product in plain English, get HTS/TARIC code suggestions.
- **CBP rulings.** 220,000+ CROSS rulings indexed and searchable.
- **REST API.** Look up codes, calculate duty, classify products, and monitor rate changes programmatically.
